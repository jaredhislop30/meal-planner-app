# Meal Planner App

This repository contains the V1 design and implementation plan for a meal planner that:

- Lets users choose meal preferences (healthy, fast, kid-friendly, comfort food)
- Lets users select meals for breakfast/lunch/dinner
- Generates recipes and a combined grocery list
- Lets users remove grocery items they already have
- Organizes grocery items by category
- Sends the accepted list to Smith's (Kroger) cart for checkout on Kroger's site
